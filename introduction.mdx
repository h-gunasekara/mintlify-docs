# Product Catalog API

The Product Catalog API allows you to search and retrieve product data from our database. Built on top of [Typesense](https://typesense.org/), a lightning-fast open-source search engine, our API enables powerful and efficient product discovery for your applications.

## Key Features

- **Full-Text Search**: Search across product titles, descriptions, vendors, and attributes
- **Filtering**: Apply precise filters based on product attributes including price, availability, color, and fabric
- **Smart Query Processing**: Automatic synonym expansion and implicit filter detection
- **Pagination**: Get paginated results for better performance and UX
- **Typesense-Powered**: Built on a modern, typo-tolerant search engine optimized for performance

## When to Use This API

Our Product Catalog API is ideal for:

- E-commerce applications needing product search functionality
- Retail dashboards and analytics tools
- Product comparison services
- Inventory management systems
- Any application that needs to display and filter product data
