## How do you get started?

### Step 1: Get your API key

To access the Catalog API, you'll need an API key. [Contact our team](mailto:founders@getcatalog.ai) to get started.

### Step 2: Authenticate your requests

Include your API key in the `x-api-key` header with every request:

```bash
curl -X POST https://api.getcatalog.ai/api/search \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY"
```

### Step 3: Make your first product search

Search for products using natural language queries. The API understands context and intent:

<CodeGroup>

```bash cURL
curl -X POST https://api.getcatalog.ai/api/search \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{"query": "black dresses under $100"}'
```

```javascript JavaScript
const response = await fetch('https://api.getcatalog.ai/api/search', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': 'YOUR_API_KEY'
  },
  body: JSON.stringify({
    query: 'black dresses under $100'
  })
});

const data = await response.json();
console.log(data.products);
```

```python Python
import requests

response = requests.post(
    'https://api.getcatalog.ai/api/search',
    headers={
        'Content-Type': 'application/json',
        'x-api-key': 'YOUR_API_KEY'
    },
    json={'query': 'black dresses under $100'}
)

data = response.json()
print(data['products'])
```

</CodeGroup>

## How do you add filters to refine search results?

Combine natural language queries with precise filters to get exactly what you need:

<CodeGroup>

```bash cURL
curl -X POST https://api.getcatalog.ai/api/search \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "query": "long dress",
    "filters": {
      "price_max": 10000,
      "is_available": true,
      "attributes": {
        "color": ["white"]
      }
    }
  }'
```

```javascript JavaScript
const response = await fetch('https://api.getcatalog.ai/api/search', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': 'YOUR_API_KEY'
  },
  body: JSON.stringify({
    query: 'long dress',
    filters: {
      price_max: 10000,
      is_available: true,
      attributes: {
        color: ['white']
      }
    }
  })
});

const data = await response.json();
```

```python Python
import requests

response = requests.post(
    'https://api.getcatalog.ai/api/search',
    headers={
        'Content-Type': 'application/json',
        'x-api-key': 'YOUR_API_KEY'

        'query': 'long dress',

            'price_max': 10000,
            'is_available': True,

                'color': ['white']

data = response.json()
```
