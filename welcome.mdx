## Core Features

<AccordionGroup>
  <Accordion title="Semantic Search">
    Advanced natural language processing enables intuitive product discovery. Search using descriptive queries like "sustainable summer dresses" or "minimalist work bags".
  </Accordion>
  
  <Accordion title="Intelligent Filtering">
    Comprehensive filtering by price, vendor, availability, and rich product attributes including color, material, style, and occasion.
  </Accordion>
  
  <Accordion title="Real-time Results">
    Fast, real-time search results powered by Typesense with sub-100ms response times for optimal user experiences.
  </Accordion>
  
  <Accordion title="AI-Ready Format">
    Response format optimized for AI applications with structured product data, attributes, and metadata.
  </Accordion>
</AccordionGroup>
