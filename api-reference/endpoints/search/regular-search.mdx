---
title: "Search Products"
api: "POST https://catalogai.vercel.app/api/search"
description: "Search products using natural language queries with advanced filtering"
---

Search for products using intelligent semantic search with natural language understanding. The API supports complex filtering, pagination, and returns structured product data optimized for AI applications.

## Request

<ParamField header="x-api-key" type="string" required>
  Your API key for authentication
</ParamField>

### Body

  Natural language search query for semantic product discovery

</ParamField>

<RequestExample>

```bash cURL
curl -X POST https://catalogai.vercel.app/api/search \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "query": "summer dress",
    "filters": {
      "max_price": 150.00,
      "is_available": true,
      "brand": ["fwrd", "amiri"],
      "attributes": {
        "color": ["blue", "white"],
        "season": "summer",
        "gender": "women"
      }
    },
    "page": 1
  }'
```

```javascript JavaScript
const response = await fetch('https://catalogai.vercel.app/api/search', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': 'YOUR_API_KEY'
  },
  body: JSON.stringify({
    query: 'summer dress',
    filters: {
      max_price: 150.00,
      is_available: true,
      attributes: {
        color: ['blue', 'white'],
        season: 'summer',
        gender: 'women'
      }
    },
    page: 1
  })
});

const data = await response.json();
```

```python Python
import requests

response = requests.post(
    'https://catalogai.vercel.app/api/search',
    headers={
        'Content-Type': 'application/json',
        'x-api-key': 'YOUR_API_KEY'
    },
    json={
        'query': 'summer dress',
        'filters': {
            'max_price': 150.00,
            'is_available': True,
            'attributes': {
                'color': ['blue', 'white'],
                'season': 'summer',
                'gender': 'women'
            }
        },
        'page': 1
    }
)

data = response.json()
```

### Basic Search

Search for products with a simple query:

<CodeGroup>

```bash cURL
curl -X POST https://catalogai.vercel.app/api/search \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{"query": "black shoes"}'
```

```javascript JavaScript
const response = await fetch('https://catalogai.vercel.app/api/search', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': 'YOUR_API_KEY'
  },
  body: JSON.stringify({
    query: 'black shoes'
  })
});
```

</CodeGroup>
