---
title: "Search Products"
api: "POST https://catalogai.vercel.app/api/search"
description: "Search products using natural language queries with advanced filtering"
---

Search for products using intelligent semantic search with natural language understanding. The API supports complex filtering, pagination, and returns structured product data optimized for AI applications.

## Request

<ParamField header="x-api-key" type="string" required>
  Your API key for authentication
</ParamField>

<ParamField header="Content-Type" type="string" required>
  Must be `application/json`
</ParamField>

### Body

<ParamField body="query" type="string">
  Natural language search query for semantic product discovery
  
  **Examples:**
  - `"silk dress for summer wedding"`
  - `"organic cotton t-shirt"`
  - `"minimalist black handbag"`
  - `"*"` (match all products)
  
  If omitted, defaults to `"*"` which returns all products.
</ParamField>

<ParamField body="filters" type="object">
  Filter criteria to narrow down results
  
  <Expandable title="Filter Properties">
    <ParamField body="filters.price_min" type="number">
      Minimum price in cents (e.g., 2500 for $25.00)
    </ParamField>
    
    <ParamField body="filters.price_max" type="number">
      Maximum price in cents (e.g., 15000 for $150.00)
    </ParamField>
    
    <ParamField body="filters.vendor" type="string | array">
      Vendor name(s) - can be single string or array of strings
    </ParamField>
    
    <ParamField body="filters.is_available" type="boolean">
      Filter by availability status
    </ParamField>
    
    <ParamField body="filters.attributes" type="object">
      Product attribute filters
      
      <Expandable title="Attribute Filters">
        <ParamField body="filters.attributes.color" type="string | array">
          Color filter (e.g., `"red"` or `["red", "blue"]`)
        </ParamField>
        
        <ParamField body="filters.attributes.material" type="string">
          Material type (e.g., `"silk"`, `"cotton"`)
        </ParamField>
        
        <ParamField body="filters.attributes.fabric" type="array">
          Fabric types (e.g., `["cotton", "linen"]`)
        </ParamField>
        
        <ParamField body="filters.attributes.gender" type="string">
          Target gender (`"women"`, `"men"`, `"unisex"`)
        </ParamField>
        
        <ParamField body="filters.attributes.season" type="string">
          Seasonal appropriateness (`"spring"`, `"summer"`, `"fall"`, `"winter"`)
        </ParamField>
        
        <ParamField body="filters.attributes.occasion" type="string">
          Suitable occasion (`"everyday"`, `"formal"`, `"party"`, `"work"`)
        </ParamField>
        
        <ParamField body="filters.attributes.category" type="string">
          Product category (e.g., `"Women > Dresses"`)
        </ParamField>
      </Expandable>
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="page" type="number" default="1">
  Page number for pagination (1-indexed)
</ParamField>

<ParamField body="limit" type="number" default="10">
  Number of items per page (currently fixed at 10)
</ParamField>

<RequestExample>

```bash cURL
curl -X POST https://catalogai.vercel.app/api/search \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "query": "summer dress",
    "filters": {
      "price_max": 15000,
      "is_available": true,
      "attributes": {
        "color": ["blue", "white"],
        "season": "summer"
      }
    },
    "page": 1
  }'
```

```javascript JavaScript
const response = await fetch('https://catalogai.vercel.app/api/search', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': 'YOUR_API_KEY'
  },
  body: JSON.stringify({
    query: 'summer dress',
    filters: {
      price_max: 15000,
      is_available: true,
      attributes: {
        color: ['blue', 'white'],
        season: 'summer'
      }
    },
    page: 1
  })
});

const data = await response.json();
```

```python Python
import requests

response = requests.post(
    'https://catalogai.vercel.app/api/search',
    headers={
        'Content-Type': 'application/json',
        'x-api-key': 'YOUR_API_KEY'
    },
    json={
        'query': 'summer dress',
        'filters': {
            'price_max': 15000,
            'is_available': True,
            'attributes': {
                'color': ['blue', 'white'],
                'season': 'summer'
            }
        },
        'page': 1
    }
)

data = response.json()
```

## Examples

### Basic Search

Search for products with a simple query:

<CodeGroup>

```bash cURL
curl -X POST https://catalogai.vercel.app/api/search \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{"query": "black shoes"}'
```

```javascript JavaScript
const response = await fetch('https://catalogai.vercel.app/api/search', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': 'YOUR_API_KEY'
  },
  body: JSON.stringify({
    query: 'black shoes'
  })
});
```

</CodeGroup>
