---
title: "Agentic Search Mini"
api: "POST https://api.getcatalog.ai/v1/agentic-search-mini"
description: "Intelligent product discovery using Google Shopping with AI-powered analysis"
---

Discover products using intelligent search that combines AI-powered productshopping search. This endpoint looks at high-quality sources and provides enriched product data with detailed attributes.

## Request

<ParamField header="x-api-key" type="string" required>
  Your API key for authentication
</ParamField>

### Request Body

<ParamField body="query" type="string" required>
  Natural language search query for product discovery
  
  **Examples:**
  - `"sustainable midi dresses"`
  - `"luxury minimalist handbags"`
  - `"versatile blazers for work"`
</ParamField>

<ParamField body="enable_enrichment" type="boolean" default="true">
  Enable AI-powered product attribute enrichment for enhanced product data analysis. Set to `false` for faster response times if detailed attributes are not needed.
</ParamField>

## Response

<ResponseField name="products" type="array">
  Array of discovered products with enriched data (returns up to 10 results)
  
  <Expandable title="Product Properties">
    <ResponseField name="url" type="string">
      Product URL (normalized and cleaned)
    </ResponseField>
    
    <ResponseField name="title" type="string">
      Product title
    </ResponseField>
    
    <ResponseField name="price_amount" type="number">
      Product price as a number (e.g., 99.99)
    </ResponseField>
    
    <ResponseField name="price_currency" type="string">
      Currency code (e.g., "USD")
    </ResponseField>
    
    <ResponseField name="description" type="string">
      Product description
    </ResponseField>
    
    <ResponseField name="images" type="array">
      Array of product image URLs
    </ResponseField>
    
    <ResponseField name="attributes" type="object">
      Comprehensive product attributes extracted through AI analysis
      
      <Expandable title="Common Attributes">
        <ResponseField name="color" type="string">
          Primary product color
        </ResponseField>
        
        <ResponseField name="material" type="string">
          Primary material composition
        </ResponseField>
        
        <ResponseField name="category" type="string">
          Product category classification
        </ResponseField>
        
        <ResponseField name="brand" type="string">
          Brand information
        </ResponseField>
        
        <ResponseField name="features" type="string">
          Key product features
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="meta" type="object">
  Search result metadata
  
  <Expandable title="Meta Properties">
    <ResponseField name="totalItems" type="number">
      Total number of products found
    </ResponseField>
    
    <ResponseField name="urls" type="array">
      Array of all product URLs returned
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST https://api.getcatalog.ai/v1/agentic-search-mini \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "query": "sustainable midi dresses",
    "enable_enrichment": true
  }'
```

```javascript JavaScript
const response = await fetch('https://api.getcatalog.ai/v1/agentic-search-mini', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': 'YOUR_API_KEY'
  },
  body: JSON.stringify({
    query: 'sustainable midi dresses',
    enable_enrichment: true
  })
});

const data = await response.json();
```

```python Python
import requests

response = requests.post(
    'https://api.getcatalog.ai/v1/agentic-search-mini',
    headers={
        'Content-Type': 'application/json',
        'x-api-key': 'YOUR_API_KEY'
    },
    json={
        'query': 'sustainable midi dresses',
        'enable_enrichment': True
    }
)

data = response.json()
```
