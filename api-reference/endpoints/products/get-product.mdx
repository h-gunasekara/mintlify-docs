---
title: "Get Product by URL"
api: "POST https://catalogai.vercel.app/api/product"
description: "Get detailed information about a specific product by URL"
---

Retrieve comprehensive details about a specific product by providing its URL. This endpoint can also detect new vendors and automatically trigger scraping for Shopify stores.

## Request

<ParamField header="x-api-key" type="string" required>
  Your API key for authentication
</ParamField>

### Request Body

<ParamField body="url" type="string" required>
  The product URL to search for
</ParamField>

<ParamField body="enable_enrichment" type="boolean" default="true">
  Whether to enable AI enrichment for single products
</ParamField>

<ParamField body="country_code" type="string" default="us">
  ISO 2-letter country code for localization (e.g., "us", "ca", "gb", "de")
  
  **Supported Country Codes:**
  nl, ca, si, co, by, ee, no, br, us, de, es, vn, il, th, gb, ph, kg, in, ru, fr, hk, ua, jp, at, mm, my, pl, au, nz, ro, tw, mx, id, dk, ng, ch, hu, sg, sa, ae, cn, ar, cl, it, tr, lv, gh, sk, gr, eg, lu, bg, se, lt, lk, kz, hr, bd, kr, cz, fi, ie, be, pt, za
</ParamField>

<RequestExample>

```bash cURL
curl -X POST https://catalogai.vercel.app/api/product \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "url": "https://www.nike.com/t/air-force-1-07-mens-shoes-5QFp5Z/CW2288-111",
    "enable_enrichment": true,
    "country_code": "us"
  }'
```

```javascript JavaScript
const response = await fetch('https://catalogai.vercel.app/api/product', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': 'YOUR_API_KEY'
  },
  body: JSON.stringify({
    url: 'https://www.nike.com/t/air-force-1-07-mens-shoes-5QFp5Z/CW2288-111',
    enable_enrichment: true,
    country_code: 'us'
  })
});

const data = await response.json();
```

```python Python
import requests

response = requests.post(
    'https://catalogai.vercel.app/api/product',
    headers={
        'Content-Type': 'application/json',
        'x-api-key': 'YOUR_API_KEY'
    },
    json={
        'url': 'https://www.nike.com/t/air-force-1-07-mens-shoes-5QFp5Z/CW2288-111',
        'enable_enrichment': True,
        'country_code': 'us'
    }
)

data = response.json()
```
