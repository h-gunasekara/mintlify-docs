---
title: "Get API Usage Statistics"
api: "GET https://api.getcatalog.ai/api/usage"
description: "Retrieve API usage statistics for your account grouped by endpoint path"
---

Get detailed usage analytics for your API key showing request counts grouped by endpoint path. This endpoint provides insight into your API consumption patterns with configurable time periods.

## Request

<ParamField header="x-api-key" type="string" required>
  Your API key for authentication
</ParamField>

### Query Parameters

<ParamField query="period" type="string" default="30d">
  Time period for usage statistics
  
  **Valid Values:**
  - `24h` - Last 24 hours
  - `7d` - Last 7 days  
  - `30d` - Last 30 days (default)
  - `90d` - Last 90 days
  - `all` - All time usage
</ParamField>

## Response

<ResponseField name="usage" type="array">
  Array of usage statistics by endpoint path
  
  <Expandable title="Usage Statistics">
    <ResponseField name="path" type="string">
      API endpoint path (e.g., "/api/search", "/api/product")
    </ResponseField>
    
    <ResponseField name="total_calls" type="number">
      Total number of API calls made to this endpoint
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="period" type="string">
  Echo back the requested time period
</ResponseField>

<RequestExample>

```bash cURL
curl -X GET "https://api.getcatalog.ai/api/usage?period=7d" \
  -H "x-api-key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://api.getcatalog.ai/api/usage?period=7d', {
  method: 'GET',
  headers: {
    'x-api-key': 'YOUR_API_KEY'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.getcatalog.ai/api/usage?period=7d',
    headers={
        'x-api-key': 'YOUR_API_KEY'
    }
)

data = response.json()
```
