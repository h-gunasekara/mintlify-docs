# Products API

Search for products in the catalog using Typesense-powered search.

## Endpoint

<ParamField path="POST" fullWidth>
  https://catalogai.vercel.app/api/products
</ParamField>

## Authentication

<Warning>
  This endpoint requires an API key to be included in the `x-api-key` header.
</Warning>

## Request Body

<ParamField body="query" type="string" optional>
  An optional natural-language search string. When provided, the API performs full-text search across product titles, descriptions, vendors, and attributes.
</ParamField>

<ParamField body="filters" type="object" optional>
  An object containing specific criteria to filter products.
</ParamField>

<ParamField body="filters.price_min" type="number" optional>
  Minimum price (inclusive).
</ParamField>

<ParamField body="filters.price_max" type="number" optional>
  Maximum price (inclusive).
</ParamField>

<ParamField body="filters.vendor" type="string | string[]" optional>
  Filter by a specific product vendor. Can be a single string for exact match or an array of strings to match multiple vendors.
</ParamField>

<ParamField body="filters.is_available" type="boolean" optional>
  Filter by product availability. Set to `true` to show only available products or `false` to show only unavailable products.
</ParamField>

<ParamField body="filters.attributes" type="object" optional>
  Filter by nested product attributes.
</ParamField>

<ParamField body="filters.attributes.color" type="string | string[]" optional>
  Filter by product color. Can be a single string for exact match or an array of strings to match multiple colors.
</ParamField>

<ParamField body="filters.attributes.fabric" type="string[]" optional>
  Filter by fabric types. Provide an array of fabric names to include.
</ParamField>

<ParamField body="filters.attributes.category" type="string" optional>
  Filter by product category.
</ParamField>

<ParamField body="page" type="number" optional default="1">
  The page number (1-indexed) for pagination.
</ParamField>

<ParamField body="limit" type="number" optional default="10">
  The number of products per page. The API uses a fixed limit of 10 items per page.
</ParamField>

### Example Request

```json
{
  "query": "cotton shirt",
  "filters": {
    "price_min": 20,
    "price_max": 100,
    "vendor": "EcoWear",
    "is_available": true,
    "attributes": {
      "color": "Blue",
      "fabric": ["cotton", "organic_cotton"]
    }
  },
  "page": 1
}
```

## Response

<ResponseField name="products" type="Product[]">
  An array of product objects matching your search criteria.
</ResponseField>

<ResponseField name="meta" type="object">
  Metadata about the search results and pagination.
</ResponseField>

<ResponseField name="meta.totalItems" type="number">
  The total number of matching products found. This value is capped at 50, regardless of the actual total.
</ResponseField>

<ResponseField name="meta.totalPages" type="number">
  The total number of pages available.
</ResponseField>

<ResponseField name="meta.currentPage" type="number">
  The current page number (matches the requested page).
</ResponseField>

<ResponseField name="meta.pageSize" type="number">
  The number of items per page.
</ResponseField>

### Product Object

<ResponseField name="id" type="number">
  The unique identifier for the product.
</ResponseField>

<ResponseField name="title" type="string">
  The product title.
</ResponseField>

<ResponseField name="description" type="string">
  The product description.
</ResponseField>

<ResponseField name="vendor" type="string">
  The product vendor or brand name.
</ResponseField>

<ResponseField name="url" type="string">
  The URL where the product can be found.
</ResponseField>

<ResponseField name="is_available" type="boolean">
  Whether the product is available in at least one variant.
</ResponseField>

<ResponseField name="attributes" type="object">
  An object containing product attributes.
</ResponseField>

<ResponseField name="attributes.category" type="string" optional>
  The product category.
</ResponseField>

<ResponseField name="attributes.color" type="string" optional>
  The product color.
</ResponseField>

<ResponseField name="attributes.fabric" type="object" optional>
  An object mapping fabric names to boolean values indicating if the product contains that fabric.
</ResponseField>

<ResponseField name="images" type="ProductImage[]">
  An array of product images.
</ResponseField>

<ResponseField name="variants" type="ProductVariant[]">
  An array of product variants.
</ResponseField>

<ResponseField name="price_currency" type="string">
  The currency of the price (currently only "USD" is supported).
</ResponseField>

<ResponseField name="price_display" type="number">
  The display price of the product (may include discounts).
</ResponseField>

<ResponseField name="price_value" type="number">
  The original value of the product before any discounts.
</ResponseField>

### ProductImage Object

<ResponseField name="url" type="string">
  The URL of the image.
</ResponseField>

<ResponseField name="alt" type="string" optional>
  Alternative text for the image.
</ResponseField>

### ProductVariant Object

<ResponseField name="title" type="string">
  The title of the variant.
</ResponseField>

<ResponseField name="color" type="string" optional>
  The color of the variant.
</ResponseField>

<ResponseField name="image" type="ProductImage" optional>
  The primary image for this variant.
</ResponseField>

<ResponseField name="options" type="VariantOption[]">
  An array of options for this variant.
</ResponseField>

<ResponseField name="url" type="string" optional>
  The URL where this specific variant can be found.
</ResponseField>

### VariantOption Object

<ResponseField name="size" type="string">
  The size of the variant option.
</ResponseField>

<ResponseField name="is_in_stock" type="boolean">
  Whether this specific size is in stock.
</ResponseField>

<ResponseField name="price" type="ProductPrice">
  The price information for this variant option.
</ResponseField>

<ResponseField name="images" type="ProductImage[]" optional>
  Size-specific images if available.
</ResponseField>

### ProductPrice Object

<ResponseField name="currency" type="string">
  The currency of the price (currently only "USD" is supported).
</ResponseField>

<ResponseField name="display_value" type="number">
  The display price (may include discounts).
</ResponseField>

<ResponseField name="value" type="number">
  The original value before any discounts.
</ResponseField>

## Error Responses

### 400 Bad Request

```json
{
  "error": "Invalid request: Request body is missing or contains malformed JSON."
}
```

### 401 Unauthorized

```json
{
  "error": "API key required (x-api-key header)"
}
```

Or:

```json
{
  "error": "Invalid API key"
}
```

### 500 Internal Server Error

```json
{
  "error": "An unexpected error occurred while processing the search request."
}
```
