## Search API

The Search API offers intelligent product discovery for AI shopping agents using natural language queries and various filters.

### Basic Discovery

Search for products matching "cotton shirt" using natural language understanding:

<CodeGroup>

```javascript Javascript
// Search for "cotton shirt"
(async () => {
  const API_KEY = "YOUR_API_KEY"; // Replace with your actual key
  
  const response = await fetch("https://catalogai.vercel.app/api/products", {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'x-api-key': API_KEY
    },
    body: JSON.stringify({ query: "cotton shirt" })
  });

  const data = await response.json();
  console.log(data);
})();
```

```python Python
import requests

# Search for "cotton shirt"
API_KEY = "YOUR_API_KEY"  # Replace with your actual key

response = requests.post(
    "https://catalogai.vercel.app/api/products",
    headers={
        'Content-Type': 'application/json',
        'x-api-key': API_KEY
    },
    json={"query": "cotton shirt"}
)

data = response.json()
print(data)
```

```bash cURL
curl -X POST https://catalogai.vercel.app/api/products \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "query": "cotton shirt"
  }'
```

</CodeGroup>

### Using Filters

Fine-tune your AI shopping agent's discovery by vendor, availability, and price range:

<CodeGroup>

```javascript Javascript
// Filter by vendor, availability, price
(async () => {
  const API_KEY = "YOUR_API_KEY"; // Replace with your actual key
  
  const response = await fetch("https://catalogai.vercel.app/api/products", {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'x-api-key': API_KEY
    },
    body: JSON.stringify({
      filters: {
        vendor: "EcoWear",
        is_available: true,
        price_min: 20.00,
        price_max: 50.00
      }
    })
  });

  const data = await response.json();
  console.log(data);
})();
```

```python Python

    "https://catalogai.vercel.app/api/products",

        'x-api-key': API_KEY

    json={
        "filters": {
            "vendor": "EcoWear",
            "is_available": True,
            "price_min": 20.00,
            "price_max": 50.00
        }
    }
)

data = response.json()
print(data)
```


## Feed API

### Basic Feed Request

<CodeGroup>

```javascript Javascript

(async () => {
```
